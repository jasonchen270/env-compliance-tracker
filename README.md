# env-compliance-tracker

A full-stack application for tracking environmental compliance reports, with role-based access and an audit trail. The backend is a .NET Web API and the frontend is built with React and Vite.

## Prerequisites

- .NET 10 SDK
- Node 20+

## Build & run

```bash
# API (from repo root)
dotnet run --project src/EnvComplianceTracker.Api
dotnet test

# Frontend
cd client
npm install
npm run dev
```
